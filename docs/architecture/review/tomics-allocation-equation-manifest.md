## Status interpretation

- `shipped default`: live TOMICS behavior that is already exposed via `partition_policy: tomics`
- `research-only`: implemented or screened only behind the opt-in research policy
- `deferred`: traced and documented, but not yet implemented as an active architecture mode

## Architectural reading of the manifest

1. The shipped default is still anchored by Heuvelink/De Koning tomato-first logic.
2. The richest exact greenhouse buffer formalism comes from Vanthoor, but it remains research-only.
3. THORP is retained only as a subordinate greenhouse root-correction family.
4. Kuijpers governs the study architecture, not the physiological tomato default.
