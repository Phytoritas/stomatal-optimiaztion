# TOMICS-HAF 2025-2C Limitations Outline

## Dataset Scope

Only one compatible measured HAF dataset is available: `haf_2025_2c`. This blocks promotion and prevents cross-dataset validation claims.

This evidence package is not universal multi-season generalization.

## Dry-Yield Basis

Dry yield is estimated from fresh yield using fixed DMC `0.056`. It is not direct destructive dry-mass measurement.

## Allocation Evidence

Direct organ partition observations are not available. Latent allocation is inferred and observer-supported, not directly validated.

## Fruit Diameter

Fruit diameter remains sensor-level apparent expansion diagnostics. It is not used for p-values, allocation calibration, or promotion.

## Plotkit Figure Status

Plotkit figure evidence is manifest-backed. Rendered PNG evidence is pending, and figure bundles should not be described as complete until real PNGs are produced.

## Shipped Default

The shipped TOMICS incumbent and `partition_policy: tomics` remain unchanged. No shipped default change is recommended at this stage.
